# jawabu-e-learning
Welcome to Jawabu, an innovative e-learning platform designed to make education accessible, interactive, and engaging for learners of all level

FEATURES
Diverse Course Selection: A variety of subjects ranging from academics to professional development.
Interactive Quizzes: Test your knowledge with engaging quizzes and real-time feedback.
User-Friendly Interface: Easy navigation and a seamless learning experience.
Progress Tracking: Monitor your learning journey with detailed progress reports.
Community & Support: Join discussions, ask questions, and connect with fellow learners.

TECH STACK
Frontend: React, Tailwind CSS
Backend: Node.js, Express
Database: MongoDB
Authentication: Firebase / JWT
Hosting: Vercel / AWS
